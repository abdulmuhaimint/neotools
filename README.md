# Neotools

A cli tool for devs

### Installation and execution
```
npm install
npm run build
npm start
```
### Global installation
```
npm install
npm run build
npm link
```
Now you can run the following command from anywhere.
```
neotools
```

### Adding a module or submodule git repo
Add modules to `REPO_INDEX` object in `src/consts.ts`. Examples are given the file itself.
